# Лабораторна робота 1.2 — Симуляція поведінки тварин

## Вимоги

- [Node.js](https://nodejs.org/) версія 18 або вище

Перевірити версію:
```bash
node -v
npm -v
```

## Встановлення

```bash
# 1. Перейти в папку проєкту
cd animals-lab

# 2. Встановити залежності
npm install
```

## Запуск

```bash
npm start
```

## Структура проєкту

```
animals-lab/
├── src/
│   ├── events/
│   │   ├── EventEmitter.ts       — Типізований event emitter
│   │   └── AnimalEvents.ts       — Типи подій тварин
│   ├── interfaces/
│   │   └── index.ts              — Інтерфейси здібностей (ISP)
│   ├── models/
│   │   ├── Animal.ts             — Абстрактний базовий клас
│   │   ├── Cat.ts                — Кіт
│   │   ├── Parrot.ts             — Папуга
│   │   └── Snake.ts              — Змія
│   ├── care/
│   │   ├── Owner.ts              — Хазяїн
│   │   └── WildLife.ts           — Тварина на волі
│   ├── simulation/
│   │   └── TimeSimulator.ts      — Симулятор часу
│   └── main.ts                   — Демонстрація (3 сценарії)
├── package.json
├── tsconfig.json
└── README.md
```

## Сценарії демонстрації

**Сценарій 1** — Кіт з хазяйкою: годування, голод, блокування бігу, щастя від прибирання.

**Сценарій 2** — Дикий папуга: смерть від голоду через 24 год без їжі.

**Сценарій 3** — Змія: щастя, скидання стану на новий день, відписка від подій.
