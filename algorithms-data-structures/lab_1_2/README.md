# Лабораторна робота 1.2 — Хеш-таблиця

**Структура даних:** Хеш-таблиця з відкритою адресацією
**Метод хешування:** Метод множення (Кнут), константа A = 0.6180339887
**Клас:** Square (квадрат)
**Ключ:** площа квадрата

## Рівні

**level1.js** — базова хеш-таблиця без вирішення колізій. Якщо позиція зайнята — елемент не додається.

**level2.js** — хеш-таблиця з квадратичним зондуванням для вирішення колізій.

**level3.js** — розширення рівня 2: додано видалення елементів за умовою (периметр > заданого порогу).

## Запуск

```bash
node level1.js
node level2.js
node level3.js
```
