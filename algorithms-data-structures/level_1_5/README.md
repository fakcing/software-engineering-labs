# Лабораторна робота 1.5 — Алгоритми пошуку

**Клас:** Student (студент)
**Поля:** прізвище, ім'я, група, стать, номер залікової книжки

## Рівні

**level1.js** — інтерполяційний пошук у впорядкованому масиві. Масив відсортовано за статтю (чоловіки перед жінками), потім за номером залікової. Пошук виконується тільки серед чоловіків.

**level2.js** — BST з вставкою у корінь через ротації. Пошук за номером залікової книжки.

**level3.js** — Splay Tree (амортизаційне балансування). При вставці та пошуку вузол піднімається до кореня через zig/zig-zig/zig-zag операції.

## Запуск

```bash
node level1.js
node level2.js
node level3.js
```
