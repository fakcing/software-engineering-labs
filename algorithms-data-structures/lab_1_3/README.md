# Лабораторна робота 1.3 — Бінарне дерево пошуку

**Структура даних:** Бінарне дерево пошуку (BST)
**Клас:** Student (студент)
**Ключ:** студентський квиток (унікальний)
**Обхід:** у ширину (BFS)

## Рівні

**level1.js** — побудова BST та виведення всіх елементів через обхід у ширину.

**level2.js** — розширення рівня 1: пошук студентів-чоловіків 3-го курсу, що проживають у гуртожитку.

**level3.js** — розширення рівня 2: видалення знайдених студентів з дерева з обробкою всіх випадків (листок, один нащадок, два нащадки).

## Запуск

```bash
node level1.js
node level2.js
node level3.js
```
