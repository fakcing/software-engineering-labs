# Лабораторна робота 1.6 — Порівняння структур даних

**Структури:** впорядкований однозв'язний список та масив
**Операції:** вставка, лінійний пошук

## Рівні

**Рівень 1** — вставка у впорядкований однозв'язний список для N, N², N³ елементів (N=20). Демонструє квадратичне зростання часу.

**Рівень 2** — порівняння лінійного пошуку в списку та масиві для трьох розмірів.

**Рівень 3** — аналіз Best / Average / Worst case лінійного пошуку в обох структурах (N=5000).

Після виконання генерує CSV файли для побудови графіків.

## Запуск

```bash
node lab16.js
```
