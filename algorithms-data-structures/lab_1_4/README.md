# Лабораторна робота 1.4 — Алгоритми сортування

**Структура даних:** Одновимірний масив
**Клас:** Student (студент)
**Поля:** прізвище, ім'я, група, номер студентського квитка

## Рівні

**level1.js** — сортування Шелла з послідовністю Кнута (1, 4, 13, 40, ...). Ключ сортування: номер групи за зростанням.

**level2.js** — сортування Шелла зі стандартною послідовністю (n/2, n/4, ...). Ключ сортування: номер групи, потім номер квитка за зростанням.

**level3.js** — висхідне сортування злиттям (bottom-up merge sort). Ключ сортування: номер групи за зростанням.

## Запуск

```bash
node level1.js
node level2.js
node level3.js
```
